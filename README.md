## Project Description
This project controls four servo motors using Arduino in Tinkercad.

The servo motors perform two actions:
1. Sweep movement for 2 seconds.
2. Stop and hold at 90 degrees.

## Components
- Arduino Uno
- 4 Servo Motors
- Jumper Wires

## Connections
| Servo | Pin |
|-------|-----|
| Servo 1 | D9 |
| Servo 2 | D6 |
| Servo 3 | D5 |
| Servo 4 | D3 |

All servo motors are connected to:
- 5V
- GND

## Software
- Tinkercad Circuits
- Arduino (C++)

## How It Works
1. The four servo motors start moving using the Sweep example.
2. The movement continues for 2 seconds.
3. After 2 seconds, all servo motors stop and hold at 90°.
